Doctumentatie Angular

INTRODUCERE

Angular este un cadru modern construit în întregime în TypeScript și, ca rezultat, utilizarea TypeScript cu Angular oferă o experiență perfectă. Documentația Angular nu numai că acceptă TypeScript ca cetățean de primă clasă, dar îl folosește ca limbaj principal. Având în vedere acest lucru, site-ul Angular va fi întotdeauna cea mai actualizată referință pentru utilizarea Angular cu TypeScript.

TOOL-URI

O serie de comenzi Angular CLI rulează un proces complex pe codul dvs., cum ar fi listing, construirea sau testarea. Comenzile folosesc un instrument intern numit Architect pentru a rula constructori CLI, care aplică un alt instrument pentru a îndeplini sarcina dorită. Cu versiunea Angular 8, API-ul CLI Builder este stabil și disponibil pentru dezvoltatorii care doresc să personalizeze Angular CLI prin adăugarea sau modificarea comenzilor. De exemplu, puteți furniza un constructor pentru a efectua o sarcină complet nouă sau pentru a schimba instrumentul terță parte utilizat de o comandă existentă.

COMPONENTE

Componentele sunt blocul principal pentru aplicațiile Angular.
Fiecare componentă este formată din:
Un șablon HTML care declară ceea ce se redă pe pagină
O clasă TypeScript care definește comportamentul
Un selector CSS care definește modul în care componenta este utilizată într-un șablon
Opțional, stiluri CSS aplicate șablonului
Acest subiect descrie cum să creați și să configurați o componentă Angular.
