Doctumentatie CSS

ANIMATII

Animațiile CSS este un modul CSS care vă permite să animați valorile proprietăților CSS în timp, folosind cadre cheie. Comportamentul acestor animații de cadre cheie poate fi controlat prin specificarea funcției de sincronizare, a duratei, a numărului de repetări și a altor atribute.

TRANSFORM

Funcția CSS translate() repoziționează un element în direcțiile orizontale și/sau verticale.
Funcția CSS rotate() definește o transformare care rotește un element în jurul unui punct fix pe planul 2D, fără a-l deforma

FLEXIBLE BOX

CSS Flexible Box Layout este un modul CSS care definește un model de casetă CSS optimizat pentru proiectarea interfeței cu utilizatorul și aspectul articolelor într-o singură dimensiune. În modelul flex layout, copiii unui container flex pot fi așezați în orice direcție și își pot „flexa” dimensiunile, fie crescând pentru a umple spațiul nefolosit, fie micșorându-se pentru a evita revărsarea părintelui. Atât alinierea orizontală, cât și verticală a copiilor pot fi ușor manipulate.

FONT

Fonturi CSS este un modul CSS care definește proprietățile legate de fonturi și modul în care sunt încărcate resursele fonturilor. Vă permite să definiți stilul unui font, cum ar fi familia, dimensiunea și greutatea acestuia, înălțimea liniei și variantele de glif de utilizat atunci când sunt disponibile mai multe pentru un singur caracter.

SELECTORI

În CSS, selectoarele sunt modele folosite pentru a selecta elementul(ele) pe care doriți să le stilați.
