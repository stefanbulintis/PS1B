Doctumentatie HTML

INTRODUCERE

HTML (Hypertext Markup Language) este un limbaj creat în scopul de a descrie, în mod text, formatul paginilor Web; fisierele create în acest limbaj vor fi interpretate de navigatoare, care vor afisa paginile în forma doritã (cu texte formatate, liste, tabele, formule, imagini, hiperlegãturi, obiecte multimedia etc.). HTML a apãrut ca o aplicatie ISO standard (apartine standardului SGML - Standard Generalized Markup Language, specializat pentru hipertext si adaptat la Web).

FORM

Formulare. HTML 1.0 permitea uzual doar transferarea informatiilor de la furnizori cãtre utilizatori, transferul în sens invers fiind foarte dificil. Odatã cu dezvoltarea Web-ului si mai ales cu utilizarea sa în scopuri comerciale si în diverse alte domenii, s-a fãcut simtitã necesitatea comunicãrii în dublu sens pentru preluãri de comenzi, completarea unor fise de înregistrare, distribuirea de produse soft, administrarea de chestionar, transmiterea unor informatii personale etc. Acestea au fost motivatiile introducerii formularelor în HTML 2.0.

MODAL

Modal este o fereastră pop-up receptivă folosită pentru a afișa conținut suplimentar. Aceasta include solicitări, configurări, consimțământ pentru cookie-uri etc. Am folosit Bootstrap si m-am documentat direct de pe site

BUTOANE

Acestea sunt elemente interactive activate de un utilizator cu un mouse, tastatură, deget, comandă vocală sau altă tehnologie de asistență. Odată activat, efectuează apoi o acțiune programabilă, cum ar fi trimiterea unui formular sau deschiderea unui dialog. În mod implicit, butoanele HTML sunt prezentate într-un stil asemănător platformei pe care rulează agentul utilizator, dar puteți schimba aspectul butoanelor cu CSS.

TAG-URI

Etichetele HTML sunt ca cuvintele cheie care definesc modul în care browserul web va formata și afișa conținutul. Cu ajutorul etichetelor, un browser web poate distinge între un conținut HTML și un conținut simplu. Etichetele HTML conțin trei părți principale: eticheta de deschidere, eticheta de conținut și eticheta de închidere. Dar unele etichete HTML sunt etichete neînchise. Când un browser web citește un document HTML, acesta îl citește de sus în jos și de la stânga la dreapta. Etichetele HTML sunt folosite pentru a crea documente HTML și pentru a le reda proprietățile. Fiecare etichetă HTML are proprietăți diferite.
